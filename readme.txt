# Cipher Vault

Cipher Vault is a simple password management application built with ASP.NET Core and Bootstrap.

## Table of Contents
- [Features](#features)
- [Getting Started](#getting-started)
- [Prerequisites](#prerequisites)
- [Installation](#installation)
- [Usage](#usage)
- [Acknowledgments](#acknowledgments)

## Features

- User authentication and session management.
- Secure password storage using encryption.
- Add, view, update, and delete password records.
- Responsive and user-friendly interface.
- Integration with FontAwesome for icons.

## Getting Started

Follow the instructions below to set up and run the project on your local machine.

### Prerequisites

- [.NET Core SDK](https://dotnet.microsoft.com/download) - Ensure you have the appropriate .NET Core SDK installed.
- [Visual Studio](https://visualstudio.microsoft.com/) or any code editor of your choice.

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/yourusername/CipherVault.git
    ```

2. Open the project in Visual Studio or your preferred code editor.

3. Build the solution.

4. Run the application.

    ```bash
    dotnet run
    ```

5. Open your web browser and navigate to [http://localhost:5000](http://localhost:5000).

## Usage

1. Register or log in to your account.
2. Add new password records by clicking on "Add New Password."
3. View, update, or delete password records on the dashboard.
4. Log out when done.
